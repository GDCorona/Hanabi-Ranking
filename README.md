# 🎆 Hanabi Ranking - Where my deepest self unfolds
### Visit the site here: https://hanabiranking.work.gd 
This is my heartfelt project whose first ideas have been established during my high school years. Since then it has gone through two major phases: the first version featured only the demonlist, while the second one introduced the other three pages. Now after a long period of procrastination and months of hard work I can finally present to you this massive new update!

## 📃 What's new
### ⭐ Features
- No new page, but the design has been significantly upgraded.
- Added dedicated info page for each demon on the demonlist, along with two sorting modes and legacy list.
- Expanded About page with a lot more juicy content.
- Navigation bar got polished and compatible on mobile.
- Added some bonus features such as footer, comments and visitor count.
### 🛠️ Code
- Huge migration from basic HTML/CSS/JS to React + Tailwind CSS.
- Integrated backend using Nodejs + MongoDB database for comments and visitor count.
- Site hosting moved from Github Pages to Render.

## ❔ Upcoming features
- **Anime Ranking:** Rank my most favorite animes with scores across several categories and detailed explaination.
- **Song Ranking:** A curated collection of my favorite songs, organized by categories such as artist or genre.
- **Stand Ranking (suggested by brother):** A list of my favorite Jojo Stands.
